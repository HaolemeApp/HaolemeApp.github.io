为什么用好了么

终端离你很远，状态离你很近。

为训练任务、远程脚本、批处理和长时间运行的命令而做。

实时运行状态

运行中、成功、失败一眼看清。设备在线状态、GPU 和 CPU 信息也在同一处。

running succeeded failed

查看运行输出

在手机上查看终端输出、搜索关键词，需要时再加载更早记录。

结束时通知

不必守着 SSH。命令完成或失败后，手机会告诉你。

项目与设备

按项目归档运行记录，在多台电脑和服务器之间快速切换。

断网后继续同步

网络恢复后补传运行记录，本地缓存让历史结果随时可看。

敏感内容端到端加密

命令、路径和控制台内容在设备端加密。云端只负责转发密文。

快速开始

三步，手机开始看终端。

01

安装 CLI

pip install -U haoleme
02

扫码配对

hao login
03

运行命令

hao echo hello

就是这么简单

只在原命令前加 hao。

命令照常在当前终端运行，输出同时安全同步到 App。

# Python training

$ hao python train.py

# Shell script

$ hao bash run.sh

# Environment variables

$ hao CUDA_VISIBLE_DEVICES=0 python train.py
